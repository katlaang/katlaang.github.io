jQuery Quiz 1


1. In jQuery terminology, what is the difference between the “jQuery function” and the “jQuery object”?

A jQuery function lets you select HTML elements using css selectors. When you call the function and pass a selector to it, it creates a new jQuery object matching all the elements that match the selector.
A jQuery object has properties and methods on it,collects and wraps the DOM elements so they can be manipulated.
