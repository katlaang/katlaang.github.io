1. What is the difference between the following 2 statements?

setTimeout(booyah, 2000);
setTimeout(booyah(), 2000);

Answer:

- setTimeout(booyah, 2000); -> call booyah function after 2000 milliseconds.
- setTimeout(booyah(), 2000); -> it calls the function immediately, rather than waiting the 2000ms!

4. What is "Unobtrusive Javascript"? What is the practical application of Unobtrusive Javascript (and the reasons for using it)? The purpose of request dispatching is to internally pass the request processing to another local resource or to tell client browser to issue another HTTP request to a specific URL.

Answer:

To separate JavaScript from HTML markup, as well as keeping modules of JavaScript independent of other modules. All content should be available without all or any of the JavaScript running successfully. Unobtrusive JavaScript should not degrade the accessibility of the HTML, and ideally should improve it, whether the user has personal disabilities or are using an unusual, or unusually configured, browser.
