Question 1: What is the main value of Java Server Pages?

JSP separates display from processing, i.e., separate html from java. JSP is html with a little java mixed in.

Question 2: How are JSP pages related to servlets?

Server converts the JSP into a servlet class that implements the javax.servlet.jsp.JspPage interface. The JSP file is compiled only when the page is first requested, or when the JSP file has been updated and has a more recent timestamp.

Question 3: How are JSP pages related to HTML?

JSP already contains HTML which means JSP allows us to include HTML.

Question 4: Give an example of a JSP scriptlet and show how it will look in the JSP servlet .

<% int value = 10; %> It will be considered as local variable in the service method. The JSP scriplet tag can only be used to declare variables and not methods. The declaration of scriptlet tag is placed inside the _jspService() method.

Question 5. Give an example of a JSP declaration and show how it will look in the JSP servlet.

<%! data value = 50; %>
It will be considered as globel variable in the class. A declaration tag can be used to declare variables as well as methods, and is placed outside of the _jspService() method.

Question 9. Explain how an EL expression is evaluated.

The JSP implementation evaluates the EL expression, converts it, and passes the returned value to the tag handler.
${something} - container evaluates this as follows
When the JSP compiler sees the ${} form in an attribute, it generates code to evaluate the expression and substitues the value of expresson.It will check in the all the scope respectively page, request, session, application. If it not found it will be ignored.

Question 10. Explain how servlet attributes are involved in EL expressions

There is four types of scopes. EL expressions are involved in four scopes.
1- Page Scope.It helps in getting the attribute stored in Page scope
2- Request Scope.It used for getting the attributes from request scope. The attribute which are set by request implicit object.
3- Session Scope. Fetches attributes from session scope, set by session object.
4- Application Scope.Used for getting Applicaton level attributes.

Question 11: Explain how servlets and JSPs use request dispatch to interact.

We use dispatch to send data as request and response to other jsp. The RequestDispatcher interface provides two methods:
1. public void forward: Forwards a request from a servlet to another source like a jsp.
2. public void include: includes the content of a resource
